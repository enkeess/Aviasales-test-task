# noAviasales

## Stack

-   `React`, `TypeScript`, `@reduxjs/toolkit`, `eslint`, `stylelint`, `plettier`
-   Load indicator display, server error handling

## Install

```sh
	git clone https://github.com/enkeess/Aviasales-test-task
	cd Aviasales-test-task
	npm i
```

## Run

```sh
	npm run dev
```
